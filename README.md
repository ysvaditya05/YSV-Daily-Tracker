# YSV Daily

YSV Daily is a personal productivity and habit tracking application built using Flutter.

The goal of the project is to provide a completely local, lightweight daily tracker that can track any measurable habit without requiring an online account or cloud storage.

---

## Current Features

### Tracker Types

- Time Tracker ✅
- Number Tracker (planned)
- Checklist Tracker (planned)
- List Tracker (planned)

---

## Time Tracker

Current functionality:

- Running timer
- Daily goals
- Progress bar
- Manual session logging
- Session deletion
- Expandable session history
- Weekly progress graph
- Weekly totals
- Local SQLite persistence

---

## Data Storage

YSV Daily stores all data locally using SQLite.

No account is required.

No cloud synchronization is currently implemented.

Future versions will support exporting monthly backups.

## Philosophy

YSV Daily is designed around a few simple principles:

- Local-first
- Fast
- Minimal
- Flexible
- No unnecessary features
- User-defined tracking instead of predefined habits

The application is intended to grow into a single place to track any daily activity, whether that is studying, sleeping, exercise, diet, reading, finances, or research.
